## <div align="center">LEIA ANTES DE ABRIR O HTML</div>
Um dos arquivos que compõe o site era grandes demais para subir no GitHub.\
Por isso, foi necessário dividir esse arquivo em 8 arquivos .ZIP.

- ### **Faça o seguinte caminho:**</br>
  _`images/logos/extrair`_

- ### **E as seguintes intruções:**</br>
  *1. Extraia o arquivo "logo1.zip" na pasta "logos" (ou seja, fora da pasta "extrair");*\
  *2. Exclua a pasta "extrair".*

### ***Após essas intruções o HTML pode ser executado sem problemas.***
